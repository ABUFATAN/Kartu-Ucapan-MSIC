# Asset yang dipakai di Home

Simpan file gambar ke folder `assets/` dengan nama berikut agar langsung terbaca oleh halaman Home:

- `image-1-ilustrasi-idul-fitri.png`
- `image-2-branding-msic.png`
- `image-template-kartu-1.png`
- `image-template-kartu-2.png`
- `image-template-kartu-3.png`
- `image-template-kartu-4.png`

Jika template kartu hanya satu file, duplikasi file yang sama menjadi 4 nama template di atas.
